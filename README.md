![alt text](http://indy-php.com/assets/img/indyphp.png)

PHP framework without dependencies, faster development applying just the essentials.<br>
No external packages or libraries are included. 


### Components

 - Custom routing
 - Mvc pattern
 - View templating
 - Csrf tokens
 - Querybuilder
 - Middlewares
 - Validation
 - Auth
